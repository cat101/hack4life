Steps

  • Install a scan agent on the router
    • Created this custom configuration "default.wl700ge"
    • # Default Configuration for the Asus wl700ge router.
      Recursive no
      Hash md5
      FollowLinks no
      IncludeAll
      <System>
      Include mod_users
      Include mod_groups
      </System>
      <Directory />
      Recursive yes
      NoEntry dev
      NoEntry proc
      NoEntry tmp
      NoEntry var
      NoEntry hdd
      NoEntry lost+found
      NoEntry opt/var
      IncludeAll
      </Directory>
  • Install a management console on a windows computer
  • Install the Osiris CLI on the same windows machine
    • osiris.exe
