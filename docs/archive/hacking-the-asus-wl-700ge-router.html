I installed the Kamikaze 7.09 firmware following this instruction set. These pages show some of the customizations that I have done to the original setup over time.

  • Monitoring the router's health
  • Getting the disk to spin down when not used
  • Backing up settings
  • New /etc/init.d/rcS that enables proper shutdown (synchronously)
  • Added Fail2Ban to monitor attacks on the ssh daemon (i.e. dropbear)
    • Fail2Ban adds firewall rules and send me emails
  • Adding integrity monitoring
  • Adding a webcam
  • Sending mail from the router
