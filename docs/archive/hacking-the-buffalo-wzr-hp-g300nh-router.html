Upgrade to barrier breaker 14.07

  • Upgrade using LUCI and the sysupgrade image
    • openwrt-ar71xx-generic-wzr-hp-g300nh2-squashfs-sysupgrade.bin
  • Install packages
    • opkg update
    • opkg install nano
    • opkg install diffutils

    • opkg install luci-app-ddns curl

      • mkdir -p /etc/ssl/certs/

      • wget -P /etc/ssl/certs/ http://curl.haxx.se/ca/cacert.pem

    • opkg install luci-app-upnp

    • opkg install wireless-tools
    • opkg install luci-ssl    
  • Settings & conf files
    • Allow local hosts to connect to SSH
    • copy ethers & hosts
    • Copy from backup
      • cp -a mybin/* /;chown -R root.root /mybin
      • cp -a www/* /www/
        • First remove this
        •  rm www/index.html
        •  rm -rf www/luci-static/
      • Do a diff with every /etc/config file
        • /config/wireless
        • /config/dropbear
        • cp dropbear/* /etc/dropbear/
    • Install MyHouse
      • opkg install haserl
      • Enable WOL
        • opkg install fping etherwake
        • chown root.root /www/cgi-bin/iphone/WOL.sh
        • chmod +s /www/cgi-bin/iphone/WOL.sh
  • NFS to NAS
    • opkg install kmod-fs-nfs nfs-utils
    • mkdir /mnt/nas; mkdir /mnt/nas_flash/
    • copy /etc/init.d/mount_nfs and do /etc/init.d/mount_nfs enable
      • tried fstab, lUCI mounts and block-mount pkg but they are not ready for NFS mounts
