Roomba power supply 220v mod (2012)

This mod was executed using Rick's page. My Roomba is the 561 and the charger's model number is 17062. The internal PCB shows STS-IRD-120-R3F-AG 08/23/2012

 

I executed the mod using the following parts from Mouser

  • 47uF 400V (EEU-EB2G470SB)
  • SS-5H RADIAL TD FUSE (SS-5H-1.6A-AP)
  • Varistor 250Vrms 1750A (V07E250P)
